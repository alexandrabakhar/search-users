# Интерфейс поиска пользователей GitHub
Позволяет:
- поиск по логину;
- фильтрация по количеству репозиториев;
- при нажатии на иконку пользователя предоставляется дополнительная информация.



## Реализован с использованием
- React + TypeScript + Vite
- Styled Components
- Vitest (тесты)
- Axios
- React-redux
